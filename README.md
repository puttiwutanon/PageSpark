# PageSpark 🔥

**แอปพลิเคชันเรียนฟิสิกส์ด้วย AI สำหรับนักเรียนมัธยมปลายไทย**

PageSpark คือแอปที่เปลี่ยนภาพถ่ายหนังสือเรียนฟิสิกส์ธรรมดา ๆ ให้กลายเป็นคลิปวิดีโออธิบายบทเรียนสั้น ๆ สไตล์ 3Blue1Brown พร้อมเสียงพากย์ภาษาไทย เพียงแค่ถ่ายรูปหน้าหนังสือ ระบบจะวิเคราะห์เนื้อหาและสร้างวิดีโอแอนิเมชันอธิบายแนวคิดให้โดยอัตโนมัติ

โปรเจกต์นี้พัฒนาขึ้นเพื่อเข้าร่วมการแข่งขัน **NSC 2026 (National Software Contest)**

## แอปนี้คืออะไร

เวลาเรียนฟิสิกส์จากตำราเรียน หลายครั้งตัวหนังสือและสูตรล้วน ๆ ก็เข้าใจยาก โดยเฉพาะเรื่องที่ต้องอาศัยจินตนาการเชิงภาพ เช่น การเคลื่อนที่ แรง หรือสนามไฟฟ้า

**PageSpark** แก้ปัญหานี้ด้วยการให้นักเรียน:

1. ถ่ายรูปหน้าหนังสือเรียนฟิสิกส์ (หรือโจทย์ที่อยากเข้าใจ)
2. ส่งภาพเข้าแอป
3. รอสักครู่ให้ AI วิเคราะห์และสร้างวิดีโอ
4. ได้วิดีโอแอนิเมชันอธิบายเนื้อหานั้นแบบเห็นภาพ พร้อมเสียงพากย์ภาษาไทย ในรูปแบบแนวตั้ง (9:16) ดูง่ายบนมือถือ

แอปจึงเหมาะกับการทบทวนบทเรียน เตรียมสอบ หรือทำความเข้าใจโจทย์ที่อ่านแล้วยังไม่กระจ่าง

## ฟีเจอร์หลัก

- 📷 **ถ่ายภาพหน้าหนังสือ** แล้วให้ AI อ่านและตีความเนื้อหาให้ทันที
- 🎬 **สร้างวิดีโออธิบายอัตโนมัติ** ด้วยแอนิเมชันสไตล์ 3Blue1Brown (กราฟ สมการ แผนภาพแรง ฯลฯ)
- 🗣️ **เสียงพากย์ภาษาไทย** อธิบายเนื้อหาไปพร้อมกับแอนิเมชัน
- 🧮 **สรุปโจทย์อัตโนมัติ** แยกเป็นหัวข้อ "โจทย์กำหนดให้ / โจทย์ถามหา / แนวคิดที่ใช้"
- 📱 **ออกแบบมาสำหรับมือถือ** วิดีโอเป็นแนวตั้ง 9:16 ดูสะดวกไม่ต้องหมุนจอ

## หลักการทำงาน

เบื้องหลังแอป PageSpark มีขั้นตอนการทำงานคร่าว ๆ ดังนี้

ถ่ายภาพหน้าหนังสือ
↓
วิเคราะห์ภาพด้วย AI (Gemini Vision)
↓
แปลงเป็นโครงสร้างบทเรียน (Lesson JSON)
↓
สร้างโค้ดแอนิเมชัน (Manim) และตรวจสอบความถูกต้อง
↓
เรนเดอร์เป็นวิดีโอ + ใส่เสียงพากย์ภาษาไทย
↓
อัปโหลดและส่งวิดีโอกลับมาให้ในแอป

## เทคโนโลยีที่ใช้

**ฝั่งเซิร์ฟเวอร์ (Backend)**
- Python + FastAPI
- Google Gemini (สำหรับวิเคราะห์ภาพและสร้างเนื้อหาบทเรียน)
- Manim (ไลบรารีสร้างแอนิเมชันเชิงคณิตศาสตร์/วิทยาศาสตร์)
- ระบบแปลงข้อความเป็นเสียงภาษาไทย (Text-to-Speech)
- Firebase / Firestore และ Cloudinary (จัดเก็บข้อมูลและวิดีโอ)
- ให้บริการผ่าน Google Cloud Run

**ฝั่งแอปมือถือ (Client)**
- React Native (Expo)
- รองรับการเล่นวิดีโอและแสดงสูตรคณิตศาสตร์/ฟิสิกส์ในแอป

## วิธีติดตั้ง

แอป PageSpark เวอร์ชันสำหรับทดสอบนี้แจกจ่ายในรูปแบบไฟล์ **APK** สำหรับ Android โดยตรง (ยังไม่ได้เผยแพร่ผ่าน Play Store) ขั้นตอนติดตั้งมีดังนี้

1. **ดาวน์โหลดไฟล์ APK** จากหน้า [Releases](https://github.com/puttiwutanon/PageSpark/releases/latest) ของโปรเจกต์ หรือกดลิงก์ดาวน์โหลดโดยตรงด้านล่าง

   👉 [ดาวน์โหลด PageSpark.apk](https://github.com/puttiwutanon/PageSpark/releases/tag/v1.0.0)

2. เปิดไฟล์ APK ที่ดาวน์โหลดมา จากแอปจัดการไฟล์ หรือจากการแจ้งเตือนดาวน์โหลดเสร็จ
3. หากเครื่องขึ้นแจ้งเตือนว่า **"ไม่อนุญาตให้ติดตั้งแอปจากแหล่งที่ไม่รู้จัก" (Install unknown apps)** ให้เข้าไปที่การตั้งค่า แล้วอนุญาตให้ติดตั้งแอปจากแหล่งนี้ได้ (เมนูนี้จะโผล่ขึ้นมาให้กดตั้งค่าได้ทันทีอยู่แล้ว)
4. กด **ติดตั้ง (Install)** แล้วรอจนติดตั้งเสร็จ
5. เปิดแอป **PageSpark** จากหน้าจอโฮมได้เลย

> ⚠️ หมายเหตุ: เนื่องจากยังไม่ได้ลงใน Play Store ระบบ Android อาจแสดงคำเตือนความปลอดภัยทั่วไปสำหรับแอปที่ติดตั้งนอก Store ซึ่งเป็นเรื่องปกติสำหรับแอปที่อยู่ในช่วงทดสอบ

## วิธีใช้งาน

1. เปิดแอป PageSpark
2. เลือกถ่ายภาพหรืออัปโหลดภาพหน้าหนังสือ/โจทย์ฟิสิกส์ที่ต้องการให้อธิบาย
3. รอระบบประมวลผล (ขั้นตอนวิเคราะห์และสร้างวิดีโออาจใช้เวลาสักครู่)
4. เมื่อวิดีโอพร้อม จะสามารถเปิดดูคลิปอธิบายพร้อมเสียงพากย์ภาษาไทยได้ทันทีในแอป
